# موقع شركة الايادي المتحدة — نسخة ثابتة قابلة للتعديل

هذه نسخة MVP لموقع "شركة الايادي المتحدة" مبنية كـ HTML/CSS/JS ثابتة مع لوحة إدارة تجريبية تحفظ البيانات في localStorage.

المحتوى والهوية:
- الألوان: أسود (#0D0D0D)، ذهبي (#C8A04F)، أبيض، رمادي.
- الخط: Cairo (Google Fonts).
- اتجاه النص: RTL (العربية).
- يدعم تعديل المحتوى عبر صفحة admin.html وحفظ التعديلات في متصفحك.

الملفات الرئيسية:
- index.html — الصفحة الرئيسية + صفحات الأقسام مدمجة.
- styles.css — تنسيق الهوية البصرية.
- script.js — تحميل البيانات من localStorage وعرضها (مشاريع، شركات، إحصاءات).
- admin.html — لوحة تحكم واجهة (mock) لتعديل البيانات وحفظها محلياً.
- admin.js — منطق حفظ واسترجاع البيانات من localStorage.

كيفية التشغيل محلياً:
1. فك الضغط أو انسخ الملفات في مجلد.
2. افتح `index.html` في المتصفح لعرض الموقع.
3. افتح `admin.html` لتعديل المحتوى وحفظه (السماح للمتصفح بتخزين البيانات محلياً).
4. بعد الحفظ في لوحة الإدارة، اعد تحميل `index.html` لرؤية التحديثات.

لنشر على GitHub Pages:
- ضع الملفات في repo عام وفعّل GitHub Pages من إعدادات الـ repo (branch: main / root).
- أو ارفع الملفات لأي استضافة ثابتة (Netlify, Vercel, Surge...).

ملاحظات مستقبلية (اقتراحات تطوير):
- بناء Backend (Node/Express + DB) لحفظ المحتوى بشكل مركزي ومستخدمين حقيقيين.
- رفع صور وميديا على CDN أو نظام ملفات مركزي.
- إضافة واجهة تسجيل دخول آمنة وأدوار (مدير، محرر).
- تحسين SEO (صفحات منفصلة لكل مشروع، meta tags، structured data).

إن احتجت أرفع المشروع على GitHub لك أو أجهز ZIP جاهز للتحميل أقدر أسوّي ذلك — قولي تفضل شنو؟
